Vous devez développer un mini logiciel de type e-commerce en mode console qui permet à un utilisateur de :
1.	Consulter une liste de produits disponibles. 
2.	Ajouter des produits à son panier (avec quantité). 
3.	Visualiser le contenu de son panier et le total à payer. 
4.	Retirer un produit du panier. 
5.	Valider sa commande (checkout). 
6.	Ajouter de nouveaux produits (fonctionnalité administrateur). 
7.	Persister les données (produits et panier) dans des fichiers JSON. 
Le programme doit être modulaire, robuste (gestion des erreurs de saisie), et autonome (création automatique des fichiers JSON avec des données initiales si absents). 
Résultat final attendu 
Chaque stagiaire doit, à l'issue du contrôle, produire un projet fonctionnel répondant aux exigences suivantes : 
1. Code source Python opérationnel 
•	Un programme exécutable sans erreur, directement depuis la ligne de commande. 
•	Code structuré, lisible, avec des fonctions bien séparées (pas de traitement lourd dans la boucle principale). 
•	Gestion automatique des fichiers JSON (products.json et cart.json) : création avec données par défaut si absents, sauvegarde après chaque modification.
